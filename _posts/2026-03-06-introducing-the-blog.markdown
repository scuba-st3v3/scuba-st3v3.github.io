Welcome to my new blog! I've set up this space to write about security, tech projects, and whatever else catches my interest.

I decided to go with a simple [Jekyll](https://jekyllrb.com/) setup hosted on GitHub Pages. It gives me full control over the content without the overhead of a complex CMS.

## What's Coming

I plan to write about:

- **Security** - Thoughts on security topics, research, and practices
- **AI** - LLMs, tooling, agentic workflows, etc.
- **Tech Projects** - Build logs and tutorials from things I'm working on
- **Miscellaneous** - Whatever else I find interesting

The blog uses the minima theme, which keeps things clean and readable.
